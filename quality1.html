Подготовка к сертификации СМК

Услуги адресованы:

клиентам, которым необходимо внедрить систему менеджмента с «нуля» и в кратчайшие сроки, в том числе для целей сертификации в
российской системе сертификации или в иностранном органе по сертификации.

 

Порядок оказания услуги:

• обследование действующей системы, проведение бизнес – анализа производственных процессов в организации клиента (беседы с сотрудниками, изучение применяемой в организации документации)
• разработка документации системы менеджмента по форме, предлагаемой нашими специалистами или по форме установленной клиентом;
• выявление работ, порядок выполнения которых должен быть изменен для обеспечения соответствия требованиям или для увеличения эффективности системы менеджмента, подготовка предложений по изменениям и способам их внедрения;
• участие в разработке системы мониторинга;
• проведение разъясняющих семинаров, бизнес-тренингов для сотрудников организации клиента;
• участие в организации и проведении внутренних аудитов, проведение консультаций по вопросу сбора данных для анализа системы менеджмента

Сроки реализации: от 1 мес. с момента обращения клиента.

Результат:

• единая и удобная для всех сотрудников организации клиента система менеджмента, обеспечивающая стабильность и улучшение производственных процессов;
• пакет документов максимально точно отражающий, существующую в организации клиента систему менеджмента;
• гарантия успешной сертификации системы менеджмента в выбранном организацией органе по сертификации.
